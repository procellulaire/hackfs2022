We use multiple databases and storage in the solution, it all depends on what type of transactions are taking place.

- MongoDB: user profile, transactions, app data, user files, Metadata, etc.   
           GridFS optional usage for larger files.
- Moralis Database: wallet login / web3 transactions / blockchain cahced data
- Tatum Database: wallet login / web3 transactions / blockchain cahced data
